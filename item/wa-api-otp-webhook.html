Apa itu whatsapp api ? 

Whatsapp api adalah sebuah tools dimana kamu bisa mengirim pesan ke whatsapp melalui aplikasi pihak ketiga, contoh sederhananya ketika website kita mempunyai fitur otp maka tools ini cocok untuk mengirimkan kode otp dengan whatsapp.

Kode OTP adalah singkatan dari One-Time Password yang berarti password sementara atau password satu kali pakai. Kode OTP biasanya digunakan untuk membantu proses verifikasi saat melakukan login atau transaksi, baik pada sistem komputer maupun perangkat digital lainnya.

Fitur Versi saat ini : 

  • Kirim Pesan (Text, Image, Video, Document) Menggunakan API's
  • Webhook (Message When Received.)
  • Mengambil Group dan Pesertanya sekaligus.

Webhook akan di triger ketika ada pesan masuk. jadi cocok untuk membuat auto responder, pesan otomatis.
